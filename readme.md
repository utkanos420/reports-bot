## О проекте

Использование бота для приёма заявок и обращений к системным администраторам колледжа является хорошим решением, позволяющим автоматизировать данный процесс. Целью данного проекта является автоматизация данных процессов

Данный проект решает следующие проблемы:

* **Системность** - заявки поступают системным администраторам через бота
* **Прозрачность** - заявки строятся через взаимодействие через бота: отсутствие лишней воды в заявке
* **Автоматизация** - администраторы сразу получают заявки
* **Интеграция** - связь с Google Sheets позволяет системотизировать поступающие заявки
* **Обратная связь** - бот позволяет взаимодействовать с отправляющими заявки
