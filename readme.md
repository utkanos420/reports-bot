## О проекте

Использование бота для приёма заявок и обращений является хорошим решением там, где необходимо оптимизировать данный процесс

Данный проект решает следующие проблемы:

* **Системность** - заявки(репорты) поступают администраторам бота: в ЛС и в Google Sheets
* **Прозрачность** - заявки оформляются через взаимодействие с ботом: интуитивно понятный интерфейс в мессенджере под рукой
* **Автоматизация** - заявки формируются по строгому шаблону, автоматически отправляются в нужные сервисы
* **Интеграция** - связь с Google Sheets позволяет систематизировать поступающие заявки
* **Обратная связь** - пользователи имеют возможность управлять своими репортами
